django.utils.datastructures.CaseInsensitiveMapping

class documentation

class CaseInsensitiveMapping(Mapping): (source)

Known subclasses: django.http.request.HttpHeaders, django.http.response.ResponseHeaders

View In Hierarchy

Mapping allowing case-insensitive key lookups. Original case of keys is preserved for iteration and string representation. Example:: >>> ci_map = CaseInsensitiveMapping({'name': 'Jane'}) >>> ci_map['Name'] Jane >>> ci_map['NAME'] Jane >>> ci_map['name'] Jane >>> ci_map # original case preserved {'name': 'Jane'}
