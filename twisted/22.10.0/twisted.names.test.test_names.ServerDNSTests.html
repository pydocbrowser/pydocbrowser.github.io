def namesTest(self, querying, expectedRecords): (source) ¶

Assert that the DNS response querying will eventually fire with contains exactly a certain collection of records.

Parameters
queryingA Deferred returned from one of the DNS client lookup methods.
expectedRecordsA list of IRecord providers which must be in the response or the test will be failed.
Returns
A Deferred that fires when the assertion has been made. It fires with a success result if the assertion succeeds and with a Failure if it fails.
