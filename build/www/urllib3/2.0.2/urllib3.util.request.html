def body_to_chunks(body: typing.Any|None, method: str, blocksize: int) -> ChunksAndContentLength: (source) ¶

Takes the HTTP request method, body, and blocksize and transforms them into an iterable of chunks to pass to socket.sendall() and an optional 'Content-Length' header. A 'Content-Length' of 'None' indicates the length of the body can't be determined so should use 'Transfer-Encoding: chunked' for framing instead.
