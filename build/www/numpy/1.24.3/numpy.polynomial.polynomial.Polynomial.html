numpy.polynomial.polynomial.Polynomial

class Polynomial(ABCPolyBase): (source)

Constructor: Polynomial(coef, domain, window, symbol)

View In Hierarchy

A power series class.

The Polynomial class provides the standard Python numerical methods '+', '-', '*', '//', '%', 'divmod', '**', and '()' as well as the attributes and methods listed in the ABCPolyBase documentation.

Parameters
coefPolynomial coefficients in order of increasing degree, i.e., (1, 2, 3) give 1 + 2*x + 3*x**2.
domainDomain to use. The interval [domain[0], domain[1]] is mapped to the interval [window[0], window[1]] by shifting and scaling. The default value is [-1, 1].
window

Window, see domain for its use. The default value is [-1, 1].

New in version 1.6.0.
