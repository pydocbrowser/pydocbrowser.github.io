def htmlentityreplace_errors(ex): (source) ¶

An encoding error handler. This python codecs error handler replaces unencodable characters with HTML entities, or, if no HTML entity exists for the character, XML character references:: >>> 'The cost was €12.'.encode('latin1', 'htmlentityreplace') 'The cost was &euro;12.'
