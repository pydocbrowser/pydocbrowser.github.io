def test_sendCommandWithTags(self): (source) ¶

Passing a command and parameters with a specified prefix and tags to IRC.sendCommand results in a proper query string including the specified line prefix and appropriate tags syntax. The query string should be output as follows: @tags :prefix COMMAND param1 param2

The tags are a string of IRCv3 tags, preceded by '@'. The rest of the string is as described in test_sendMessage. For more on the message tag format, see the IRCv3 specification.
