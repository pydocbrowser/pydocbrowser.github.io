class documentation

class CompareToZeroChecker(checkers.BaseChecker): (source)

View In Hierarchy

Checks for comparisons to zero. Most of the time you should use the fact that integers with a value of 0 are false. An exception to this rule is when 0 is allowed in the program and has a different meaning than None!
