def makeReconnector(self, fireImmediately=True, startService=True, protocolType=Protocol, **kw): (source) ¶

Create a ClientService along with a ConnectInformation indicating the connections in progress on its endpoint.

Parameters
fireImmediately:boolShould all of the endpoint connection attempts fire synchronously?
startService:boolShould the ClientService be started before being returned?
protocolTypea 0-argument callable returning a new IProtocol provider to be used for application-level protocol connections.
**kwArbitrary keyword arguments to be passed on to ClientService
Returns
a 2-tuple of ConnectInformation (for information about test state) and ClientService (the system under test). The ConnectInformation has 2 additional attributes; applicationFactory and applicationProtocols, which refer to the unwrapped protocol factory and protocol instances passed in to ClientService respectively.
