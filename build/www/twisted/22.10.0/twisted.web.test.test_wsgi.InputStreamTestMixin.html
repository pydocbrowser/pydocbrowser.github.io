class InputStreamTestMixin(WSGITestsMixin): (source)

Known subclasses: twisted.web.test.test_wsgi.InputStreamBytesIOTests, twisted.web.test.test_wsgi.InputStreamTemporaryFileTests

View In Hierarchy

A mixin for TestCase subclasses which defines a number of tests against _InputStream. The subclass is expected to create a file-like object to be wrapped by an _InputStream under test.
