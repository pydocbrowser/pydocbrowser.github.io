class ReturningDelete(Update, TypedReturnsRows[_TP]): (source)

Constructor: ReturningDelete(table)

View In Hierarchy

Typing-only class that establishes a generic type form of :class:`.Delete` which tracks returned column types. This datatype is delivered when calling the :meth:`.Delete.returning` method. .. versionadded:: 2.0
