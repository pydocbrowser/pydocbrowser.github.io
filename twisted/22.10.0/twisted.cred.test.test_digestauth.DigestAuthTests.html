class DigestAuthTests(TestCase): (source)

View In Hierarchy

TestCase mixin class which defines a number of tests for DigestCredentialFactory. Because this mixin defines setUp, it must be inherited before TestCase.

Method formatResponse Format all given keyword arguments and their values suitably for use as the value of an HTTP header.

Method test_caseInsensitiveAlgorithm The case of the algorithm value in the response is ignored when checking the credentials.
Method test_checkHash DigestCredentialFactory.decode returns an IUsernameDigestHash provider which can verify a hash of the form 'username:realm:password'.
Method test_failsWithDifferentMethod DigestCredentialFactory.decode returns an IUsernameHashedPassword provider which rejects a correct password for the given user if the challenge response request is made using a different HTTP method than was used to request the initial challenge.
Method test_incompatibleCalcHA1Options calcHA1 raises TypeError when any of the pszUsername, pszRealm, or pszPassword arguments are specified with the preHA1 keyword argument.
