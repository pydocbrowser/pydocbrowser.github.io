class DeprecatedSelectGenerations(object): (source)

Known subclasses: sqlalchemy.sql.expression.Select

View In Hierarchy

A collection of methods available on :class:`_sql.Select`, these are all **deprecated** methods as they modify the :class:`_sql.Select` object in -place.

Method append_column Append the given column expression to the columns clause of this select() construct.
Method append_correlation Append the given correlation expression to this select() construct.
Method append_from Append the given :class:`_expression.FromClause` expression to this select() construct's FROM clause.
Method append_having Append the given expression to this select() construct's HAVING criterion.
Method append_prefix Append the given columns clause prefix expression to this select() construct.
Method append_whereclause Append the given expression to this select() construct's WHERE criterion.
