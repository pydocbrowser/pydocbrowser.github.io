fakePlatformTrust:boolShould we fake the platformTrust to be the same as our fake server certificate authority, so that we can test it's being used? Defaults to 'no' and we just pass platform trust.
useDefaultTrust:boolShould we avoid passing the trustRoot to ssl.optionsForClientTLS? Defaults to 'no'.
