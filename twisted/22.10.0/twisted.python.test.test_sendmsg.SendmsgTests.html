Tests for the Python2/3 compatible sendmsg interface.

Method setUp Create a pair of UNIX sockets.
Method tearDown Close the sockets opened by setUp.

Method test_syscallError If the underlying sendmsg call fails, send1msg raises socket.error with its errno set to the underlying errno value.
