Loads and stores an ast to disk. Args: ast: The pytd.TypeDeclUnit to save to disk. filename: The filename for the pickled output. If this is None, this function instead returns the pickled string. open_function: A custom file opening function. is_package: Whether the module with the given ast is a package. src_path: Optionally, the filepath of the original source file. metadata: A list of arbitrary string-encoded metadata. Returns: The pickled string, if no filename was given. (None otherwise.)
