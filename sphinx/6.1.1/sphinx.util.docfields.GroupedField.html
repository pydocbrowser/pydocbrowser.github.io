sphinx.util.docfields.GroupedField

class GroupedField(Field): (source)

Known subclasses: sphinx.domains.python.PyGroupedField, sphinx.util.docfields.TypedField

View In Hierarchy

A doc field that is grouped; i.e., all fields of that type will be transformed into one field with its body being a bulleted list. It always has an argument. The argument can be linked using the given *rolename*. GroupedField should be used for doc fields that can occur more than once. If *can_collapse* is true, this field will revert to a Field if only used once. Example:: :raises ErrorClass: description when it is raised
