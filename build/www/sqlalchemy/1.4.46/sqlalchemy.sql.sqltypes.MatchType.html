sqlalchemy.sql.sqltypes.MatchType

class documentation

class MatchType(Boolean): (source)

Known subclasses: sqlalchemy.dialects.mysql.types._MatchType

View In Hierarchy

Refers to the return type of the MATCH operator. As the :meth:`.ColumnOperators.match` is probably the most open-ended operator in generic SQLAlchemy Core, we can't assume the return type at SQL evaluation time, as MySQL returns a floating point, not a boolean, and other backends might do something different. So this type acts as a placeholder, currently subclassing :class:`.Boolean`. The type allows dialects to inject result-processing functionality if needed, and on MySQL will return floating-point values. .. versionadded:: 1.0.0
