class _omd_bucket: (source)

Constructor: _omd_bucket(omd, key, value)

View In Hierarchy

Wraps values in the :class:`OrderedMultiDict`. This makes it possible to keep an order over multiple different keys. It requires a lot of extra memory and slows down access a lot, but makes it possible to access elements in O(1) and iterate in O(n).
