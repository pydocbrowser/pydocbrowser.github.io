class Connector(Dialect): (source)

Known subclasses: sqlalchemy.connectors.pyodbc.PyODBCConnector

View In Hierarchy

Base class for dialect mixins, for DBAPIs that work across entirely different database backends. Currently the only such mixin is pyodbc.
