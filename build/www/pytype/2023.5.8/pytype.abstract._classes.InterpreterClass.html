def _mangle(self, name): (source) ¶

Do name-mangling on an attribute name. See https://goo.gl/X85fHt. Python automatically converts a name like "__foo" to "_ClassName__foo" in the bytecode. (But "forgets" to do so in other places, e.g. in the strings of __slots__.) Arguments: name: The name of an attribute of the current class. E.g. "__foo". Returns: The mangled name. E.g. "_MyClass__foo".
