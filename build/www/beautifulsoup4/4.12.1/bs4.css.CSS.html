class CSS(object): (source)

View In Hierarchy

A proxy object against the soupsieve library, to simplify its CSS selector API. Acquire this object through the .css attribute on the BeautifulSoup object, or on the Tag you want to use as the starting point for a CSS selector. The main advantage of doing this is that the tag to be selected against doesn't need to be explicitly specified in the function calls, since it's already scoped to a tag.
