def parameterized_argument_value_pairs(func, p): (source) ¶

Return tuples of parameterized arguments and their values.

This is useful if you are writing your own doc_func function and need to know the values for each parameter name:

>>> def func(a, foo=None, bar=42, **kwargs): pass
>>> p = param(1, foo=7, extra=99)
>>> parameterized_argument_value_pairs(func, p)
[("a", 1), ("foo", 7), ("bar", 42), ("**kwargs", {"extra": 99})]

If the function's first argument is named self then it will be ignored:

>>> def func(self, a): pass
>>> p = param(1)
>>> parameterized_argument_value_pairs(func, p)
[("a", 1)]

Additionally, empty *args or **kwargs will be ignored:

>>> def func(foo, *args): pass
>>> p = param(1)
>>> parameterized_argument_value_pairs(func, p)
[("foo", 1)]
>>> p = param(1, 16)
>>> parameterized_argument_value_pairs(func, p)
[("foo", 1), ("*args", (16, ))]
