def test_removingLastTaskStopsScheduledCall(self): (source) ¶

If the last task in a Cooperator is removed, the scheduled call for the next tick is cancelled, since it is no longer necessary.

This behavior is useful for tests that want to assert they have left no reactor state behind when they're done.
