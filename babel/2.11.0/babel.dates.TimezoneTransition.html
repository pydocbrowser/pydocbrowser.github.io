class TimezoneTransition: (source)

View In Hierarchy

A helper object that represents the return value from :func:`get_next_timezone_transition`. This class is pending deprecation with no replacement planned in the Babel library. :field activates: The time of the activation of the timezone transition in UTC. :field from_tzinfo: The timezone from where the transition starts. :field to_tzinfo: The timezone for after the transition. :field reference_date: The reference date that was provided. This is the `dt` parameter to the :func:`get_next_timezone_transition`.
