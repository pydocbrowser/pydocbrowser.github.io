class TypeStrVisitor(SyntheticTypeVisitor[str]): (source)

Known subclasses: mypy.stubgen.AnnotationPrinter, mypy.suggestions.TypeFormatter

View In Hierarchy

Visitor for pretty-printing types into strings. This is mostly for debugging/testing. Do not preserve original formatting. Notes: - Represent unbound types as Foo? or Foo?[...]. - Represent the NoneType type as None.
